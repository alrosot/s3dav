s3DAV
=====
s3DAV is a webDAV server which allows you to access a virtual file system using your Amazon s3 account.

How to run
==========
Just type in this command :
java -jar s3DAV-0.2.jar

This will open a s3DAV server on the port 8070.

The configuration page is : http://127.0.0.1:8070/index.html

More information
================
More information can be found here: http://www.carion.org/s3dav/index.html
